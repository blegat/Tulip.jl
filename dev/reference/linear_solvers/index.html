AbstractLinearSolver

This is the base type from which all implementations should derive.

Tulip.TLPLinearAlgebra.AbstractLinearSolver — Type
AbstractLinearSolver{Tv}

Abstract container for solving an augmented system

    [-(Θ⁻¹ + Rp)   Aᵀ] [dx] = [ξd]
    [   A          Rd] [dy]   [ξp]

where ξd and ξp are given right-hand side.

source

Custom linear solvers should inherit from the AbstractLinearSolver class, and extend the following two functions:

Tulip.TLPLinearAlgebra.update_linear_solver! — Function
update_linear_solver!(ls, θ, regP, regD)

Update internal data, and re-compute factorization/pre-conditioner.

After this call, ls can be used to solve the augmented system

    [-(Θ⁻¹ + Rp)   Aᵀ] [dx] = [ξd]
    [   A          Rd] [dy]   [ξp]

for given right-hand sides ξd and ξp.

source
Tulip.TLPLinearAlgebra.solve_augmented_system! — Function
solve_augmented_system!(dx, dy, ls, ξp, ξd)

Solve the symmetric quasi-definite augmented system

    [-(Θ⁻¹ + Rp)   Aᵀ] [dx] = [ξd]
    [   A          Rd] [dy]   [ξp]

and over-write dx, dy with the result.

Arguments

  • dx, dy: Vectors of unknowns, modified in-place
  • ls: Linear solver for the augmented system
  • ξp, ξd: Right-hand-side vectors

DenseLinearSolver

Tulip.TLPLinearAlgebra.DenseLinearSolver — Type
DenseLinearSolver{Tv}

Linear solver for the 2x2 augmented system

    [-(Θ⁻¹ + Rp)   Aᵀ] [dx] = [xi_d]
    [   A          Rd] [dy]   [xi_p]

with $A$ dense.

The augmented system is automatically reduced to the normal equations system. BLAS/LAPACK functions are used whenever applicable.

SparsePosDefLinearSolver

Tulip.TLPLinearAlgebra.SparsePosDefLinearSolver — Type
SparsePosDefLinearSolver

Linear solver for the 2x2 augmented system

    [-(Θ⁻¹ + Rp)   Aᵀ] [dx] = [xi_d]
    [   A          Rd] [dy]   [xi_p]

with $A$ sparse.

Uses a Cholesky factorization of the positive definite normal equations system

(A * ((Θ⁻¹ + Rp)⁻¹ * Aᵀ + Rd) dy = xi_p + A * (θ⁻¹ + Rp)⁻¹ * xi_d
                              dx = (Θ⁻¹ + Rp)⁻¹ * (Aᵀ * dy - xi_d)
